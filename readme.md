Penetration testing suite that creates a Kali Linux VM through Vagrant for ethical hacking and security assessments.

## Tools Included

Custom pentesting tools created by CYRAX:
- **AI** - AI-powered security tools
- **Archives** - Archive analysis utilities
- **Attacks** - Attack simulation tools
- **Core** - Core penetration testing utilities
- **Databases** - Database security tools
- **Defense** - Defensive security tools
- **JWT Tokens** - JWT security analysis
- **Reconnaissance** - Information gathering tools
- **Stealth** - Stealth and evasion techniques
- **WordPress** - WordPress security testing

## Setup

### Option 1: Automatic Setup
```bash
make setup
```

### Option 2: Manual Setup
```bash
cd cyrax
vagrant up
vagrant ssh
```

## Requirements

- Make (for automatic setup)
- VirtualBox and Vagrant
- Internet connection
